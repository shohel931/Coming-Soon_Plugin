A simple plugin to show a coming soon page for your WordPress site. Visitors will see the coming soon page, while admins can view the full site.

== Description ==

**Coming Soon Mysite** is a lightweight and easy-to-use plugin that displays a customizable *Coming Soon* page to your visitors while your website is under construction. Logged-in administrators can still view the site normally.

== Frequently Asked Questions ==

= Who will see the Coming Soon page? =  
Only logged-out users (visitors). Logged-in admins will see the full site.

= Can I customize the Coming Soon page content or design? =  
Yes, you can edit the `coming-soon-template.php` file and the CSS file (`coming-style.css`) in the plugin folder.
